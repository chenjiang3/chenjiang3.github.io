选择排序

发表于 2018-05-02 | 分类于 算法数据结构 |

算法简介

相比冒泡排序，选择排序算法思想相对简单，它的工作原理如下，从未排序的序列中找到最小（或最大）的元素，存放到序列的起始位置，然后，再从剩余的序列中找到最小（或最大）的元素，放到已排序序列的末尾，以此类推，直到所有的元素都有序。

相比冒泡排序，选择排序交换元素的次数要少，n个元素的序列最多要n-1次交换。

选择排序和冒泡排序一样，时间复杂度是O(n^2)。

算法流程

流程

假设要对【5, 1, 4, 2, 8】从小到大进行排序。

1、第一步，从【5, 1, 4, 2, 8】中找到最小数，当前为1，则和5交换，序列变成【1，5，4，2，8】，【1】为有序序列，【5，4，2，8】为无序序列。

2、从【5，4，2，8】中找到最小的2，放到已经有序的序列【1】末尾，则原序列变成【1，2，4，5，8】，【1，2】为有序序列，【4，5，8】为无序序列。

3、从【4，5，8】中找到最小的4，放到有序序列后面，原序列变成【1，2，4，5，8】，【1，2，4】为有序序列，【5，8】为无序序列。

4、从【5，8】中找到最小的5，放到有序序列后面，原序列变成【1，2，4，5，8】，有序序列为【1，2，4，5】，无序序列为【8】。

5、最后把8放到有序序列后面，排序结束。

从上面的排序流程可看出，选择排序就是从无序的序列中找出最小（最大）的数，依次放到有序序列中即可。
